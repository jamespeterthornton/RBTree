Welcome, welcome -

This is a Java applet for displaying an RBTree. You can add, delete, and insert.

In order to run set up a local server. One way to do this is to open up
terminal, cd to the directory with these files in it, and type 

python -m SimpleHTTPServer 8888 &.

Then, open the html file and enjoy.
